Design Arguments

Gold-Standard Story
Tasks Points
List all project stakeholders. 0.5
Determine the design outcomes for each stakeholder class. 1
For each outcome, list the current user needs. 3
Define design needs that the solution must include in order to address the user needs. 2
For each outcome, list design arguments that detail what features the solution should include. 3
Detail how each design argument would be implemented (specify technologies/approaches). 1
